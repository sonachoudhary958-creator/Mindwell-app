# 🌿 MindWell – Mental Wellness App

## 🚀 Live Demo
https://mindwell-app.onrender.com

## 📌 Project Overview
MindWell is a full-stack mental wellness journaling web application where users can:
- Write daily journals ✍️
- Track mood 😊
- View analytics 📊
- Use breathing relaxation tool 🌬️

## ⚙️ Tech Stack
- Node.js
- Express.js
- PostgreSQL
- Chart.js
- HTML + CSS

## ✨ Features
- User Authentication (Register/Login)
- Add & Delete Journals
- Mood Tracking (1-10)
- Mood Analytics Graph
- Breathing Exercise Tool

## 💡 Future Improvements
- Encryption for journal privacy 🔐
- Dark Mode 🌙
- Edit Journal feature ✏️
